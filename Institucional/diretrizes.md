## Diretrizes

O bjetivo do projeto é criar uma aplicação web que as empresas de internet possam utilizar para pesquisar o historico dos clientes que querem contratar o serviço dela, nesse momento é provedor de internet mas pode ser qualquer tipo de empresa.
atualmente em muitos casos o mesmo cliente contrata o serviço de uma empresa e não paga, vai para outra empresa e não paga, e assim por diante.
diante desse problema, as empresas se uniram para criar esse hub centralizado.
